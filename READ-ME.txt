# AutoWash Car Wash Management System

## Requirements
- PHP 7.4+
- MySQL 5.7+
- Composer

## Installation
1. Clone repository
2. Run `composer install`
3. Copy `.env.example` to `.env` and configure
4. Import database schema
5. Configure web server

## Features
- User registration/login with Google OAuth
- Appointment booking system
- Admin dashboard
- Payment management
- Email notifications

## Security
- CSRF protection
- Password hashing
- Prepared statements
- Session management
